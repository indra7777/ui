Description

A shaping machine is a type of machine tool that uses linear relative motion between the workpiece and a single-point cutting tool to machine a linear toolpath. Its cut is analogous to that of a lathe, except that it is linear instead of rotational. The tool moves linearly relative to the workpiece in the primary cutting motion.
